  import ggplotnim

How I start

nim r toc init

In a new folder create a toc.nim file:

import nimibook

var book = initBookWithToc:
  entry("Introduction", "index.md")
  entry("Example", "example")

nimibookCli(book)

you can pick any name instead of toc (e.g nbook, ...)

nim r toc init
